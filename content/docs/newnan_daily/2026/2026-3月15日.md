# 3月15日

方教授被ban了。以下是《星海纪》的 方教授小传。

```
2月28日，wanghany另一位线下朋友 Pr_Fang，号 方教授 ，作答问卷，当是时，虽有牛腩手册，而置而不观，数询左右。莫得其旨，乃稍泛览焉。

方入服，思陆南，踏星辰，览离璇，曰：“此岂非城镇耶？”复曰：“吾昔以旬日之功，蒇素晴之《天空之城》之鸿构，必将复刻于牛腩之域！”当时是，离璇已覆灭，玩家无人上线，土地收归管理。询其选址，初对曰：“未有所定”，而后定之于离璇。虽然，以wanghany引荐故，依例隶星海之籍，而依依不忘其项目。夫作天空之城者，是为方教授所倡之首议也。

教授复议曰：“将营广陌麦畴，以证吾营构巨工之能。”wanghany、hmyqaq闻之，咸愕然莫解。教授乃申其旨曰：“找一个最有可能扩张的镇，把他封上。而且石英建筑有很多，没什么特点。构造地形是为了咱们退坑后留下痕迹。”此三句者，盖意在使星海修麦田，以遏邻镇之扩张耳。

教授复陈其三策，曰：“君素习星海之执事乎？吾思可与之结纳；吾复有一规划：当于雪山之阳阴，建双核心聚落……”半晌，复曰：“以星沙地区之在线率不高者，当谏其主：平地星沙、修路通达。盖其境势将衰，乃众人皆知，纵不鼎新革故，亦当先治道途。”

而后于西北 (-4398, -4126) 建其基地，后取炸药若干，于离璇原教堂附近作混凝土固化机，又欲修刷石机，言工业区修毕即申请聚落。曰：“重铸离璇荣光，吾辈义不容辞。”

然其固化机被判“违建”，又询问改名之事，因被告知“弗可改名”而鉴政群中，为管理所ban，其西北基地亦为NSrank所平。

R.I.P.
```
